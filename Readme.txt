Proyecto: Aplicación de venta online de una dietetica

Autor: Nicolas Gonzalez
Fecha: 29/03/2024
Versión: 1.0

Objetivo:

El objetivo del proyecto es permitir la gestion de una pagina que contenga los clientes, vendedores, productos y productos al por mayor
de una dietetica. 
En esta pagina se podra crear, modificar, borrar y visualizar lo antes mencionado. 
Ademas, se agrego una funcion search para los productos (no para los productos al por mayor).
Todas estas funciones solo se mostraran si el usuario se encuentra logueado. Si no tiene usuario se le dara la opcion de registrarse.
Por otro lado una vez logueado el usuario podra modificar su perfil y su foto de perfil.
Ademas de eso en la parte inferior de la pagina hay un link denominado "acerca de mi" que cuenta con mi informacion personal.


Modelos usuados:

- Vendedores: se solicita el nombre, apellido, email y DNI para poder crear un vendedor nuevo 
- Clientes: se solicita el nombre, apellido, email y DNI para poder crear un cliente nuevo.
- Productos: se solicita el nombre, el peso en gramos y el precio en pesos para poder crear un producto nuevo.
- Productos por mayor: se solicita el nombre, el peso en kg y el precio en pesos para poder crear un producto al por mayor nuevo.

Pruebas: Login, Registro de usuario, Hacer click a los links, Carga de productos, Carga de clientes, Edicion de vendedores,
Eliminacion de productos que se vendan al por mayor, Edicion del perfil del usuario, Logout, Login requerido
